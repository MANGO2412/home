¿QUE ES JAVASCRIPT?

publicacion: 21/11/2021

JavaScript es un lenguaje de programación de tipo interpretado,” ¿qué quiere decir con lenguaje interpretado?” quiere decir,que el código escrito en este lenguaje o en estos tipos de lenguajes, no necesita convertirse a lenguaje máquina por un software(compiladores), para que tu programa se ejecute, de hecho en javaScript la ejecución es al instante. JavaScript es utilizado mayormente en la web, permite construir o desarrollar sitios web dinámicos e interactivos para el usuario junto a otros dos lenguajes informáticos, que son HTML(“ se utiliza para el desarrollo de la estructura del sitio web”) y CSS(“se utiliza para el diseño del sitio web”).

EL ORIGEN DE JAVASCRIPT Y ECMA

JavaScript fue desarrollado por el programador Brandan Erich en septiembre de 1995, el lenguaje se desarrolló por tan solo diez días, su nombre original fue mocha después lo cambiaron a LiveScript y más tarde fue nombrado como JavaScript. Por ese mismo año, había una guerra tecnológica entre dos navegadores, Netscape y Explorer el navegador de Microsoft, JavaScript fue desarrollado en Netscape, como Netscape estaba perdiendo la Batalla con Microsoft, ellos decidieron buscar una organización que se encarga de actualizar y estandarizar el lenguaje,la organización que se encargó y que actualmente está a cargo de este lenguaje, es The European Computer Manufacturers (ECMA), ellos actualizan el lenguaje y cada versión de JavaScript es denominada EcmaScript con el número de versión del lenguaje

JavaScript vs Java

Hay personas que confunden JavaScript con Java, debido a que poseen el mismo nombre y creen que sus sintaxis puede ser similar, lo cierto es que,¡no son iguales!, esos lenguajes poseen una gran diferencia en la ejecución y sintaxis del código, en Java el código escrito se tiene que convertir a lenguaje máquina mediante un compilador y después ejecutar el código, en JavaScript solo Escribes código y al instante lo ejecutas a través de un navegador como Chrome,Firefox y etc

Ahora, te preguntaras, si no son iguales ¿por JavaScript tienen el nombre de Java del otro lenguaje? Como JavaScript se Desarrolló en Netscape, entonces Netscape tenía soporte de java, por eso decidieron poner el nombre de JavaScript y además, que eso fue una estrategia de marketing para llamar la atención de los programadores.

¿Que se puede hacer en JavaScript aparte del desarrollo web?

Al inicio JavaScript no se consideraba como un excelente lenguaje,debido a que su desarrollo fue en poco tiempo, JavaScript tenía algunas limitaciones al programar,entonces los programadores consideraban a JavaScript como un lenguaje para diseño y para los diseñadores, a través de los años y con buenas o malas actualizaciones esta idea que se tenía de JavaScript, cambio.

Actualmente JavaScript es el lenguaje mayor utilizado, ya que no se enfoca solo en el front-end(diseño de sitios web) También en el Back-end(“la conexión de servidores con el sitio web”) sin olvidar también las apis, frameworks y librerías,que son herramientas escritas en JavaScript que nos permite desarrollar otras cosa mas interesante que solo sitios web dinamicos. En JavaScript se puede desarrollar videojuegos, aplicaciones móvil, aplicaciones web y aplicaciones de escritorio, esto se puede desarrollar solo con fundamentos web y uso de HTML, CSS, JavaScript y con la ayuda de algunas herramientas ya mencionadas.

UN MENSAJE PARA EL PROGRAMADOR PRINCIPIANTE

Si tu eres un programador principiante como yo y estas en busca un lenguaje para aprenderlo,como me dijo un sabio programador: “para escoger un lenguaje, tienes que definir qué es lo que quieres construir, para después investigar qué lenguaje te permite desarrollar eso que tienes en mente y más después buscar contenido que te permita aprender el lenguaje” Si quieres desarrollar algo relacionado a web, JavaScript es tu principal opción, pero antes de adentrarte en lenguaje, Tendrás que aprender HTML y CSS.
